# Getting and Cleaning Data - Course Project


### Modifications made to the original data

* The training and test dataset where merged
* Extracted only the measurement related to mean and standard deviation
* Applied descriptive names to the activities in the data set
* Applied desciptive variable names to the data set
* Created a tidy data set with the average of each variable for each activity and subject


### Descriptions

**Identifiers**
The first two columns are Identifiers:
* Subject: the ID of the Subject
* Activity: Name of the activity performed by the subject
